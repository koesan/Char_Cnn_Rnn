# Char-Cnn-Rnn

Bu projenin amacı, kendi veri setimizi kullanarak Char-CNN-RNN modelini geliştirmek ve verilerimizi gömmektir (embedding).

## Gereksinimler

öncelikle elinide txt ve img verileri bununması gerkeiyor

### Veri Seti Klasör Yapısı

```
Dataset/
├── text/
│   ├── dosya1.txt
│   ├── dosya2.txt
│   ├── dosya3.txt
│   └── ...
└── img/
    ├── resim1.jpg
    ├── resim2.jpg
    ├── resim3.jpg
    └── ...
```
